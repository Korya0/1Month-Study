# Advanced Flutter Architecture Curriculum

هذه هي الخطة الشاملة لتعلم معمارية فلاتر المتقدمة (Advanced Flutter Architecture).

## Lectures

- **Lecture 1**: The Three Trees: How Flutter Renders UI Under the Hood (شجرة الوجيستس الثلاثية ومفهوم Declarative UI)
- **Lecture 2**: State Management Foundations: setState vs InheritedWidget (الأساس الذي تُبنى عليه كل مكتبات إدارة الحالة)
- **Lecture 3**: Dependency Injection: Decoupling Code with GetIt (حقن التبعيات وتقليل الاعتمادية Loose Coupling)
- **Lecture 4**: Event-Driven State: Mastering the BLoC Pattern (إدارة الحالة بالأحداث باستخدام BLoC)
- **Lecture 5**: Clean Architecture: The Domain Layer and Use Cases (الطبقة المستقلة وتطبيق مبادئ SOLID)
- **Lecture 6**: The Data Layer: Repositories and Error Handling (نمط المستودع وتوحيد الأخطاء باستخدام fpdart)
- **Lecture 7**: Network & Local Storage: Dio Interceptors and Hive (إدارة الاتصال بالشبكة واستراتيجيات الكاش)
- **Lecture 8**: Routing Architecture: Declarative Navigation with GoRouter (الراوتينج المتقدم والحماية Route Guards)
- **Lecture 9**: Modular Architecture: Feature-First Folder Structure (هيكلة الملفات حسب الميزات للمشاريع الكبيرة)
- **Lecture 10**: Testing the Architecture: Unit, Widget, Integration, and Mocking (أنواع الاختبارات الثلاثة واستخدام Mocktail)
- **Lecture 11**: Final Mini-Project: Offline-First App with Clean Architecture (تطبيق استراتيجية Offline-First باستخدام Hive و Dio)
